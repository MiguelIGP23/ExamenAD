# Examen Acceso a Datos (RA2) - 1,5 horas
## Contexto: Gestión de empresas de desarrollo y alumnos en prácticas

Completa los `// TODO` para construir un pequeño sistema de gestión de empresas y alumnos.
Tendrás que arreglar las trampas que encuentres por el camino. Es decir, no solo hay que 
escribir código en las zonas de TODO.

### Funcionalidad requerida
- CRUD parcial:
  - **Empresa:** listar, insertar, modificar
  - **Alumno:** listar con JOIN, insertar, eliminar
- Todo aquello que está en la rúbrica aportada en clase:
    - Patrón DAO + Singleton + división en capas
    - Sin malos olores (comentarios, nombres claros, métodos largos, código duplicado)
- Añadir: 
    - Transacción en una de las opciones del menú
    - Añadir un método para ejecutar un procedimiento almacenado. Ya que Sqlite no los soporta, dejarlo comentado.
